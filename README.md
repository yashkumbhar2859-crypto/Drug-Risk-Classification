#  Drug Regulatory Risk Classification System

##  Project Overview
This project focuses on classifying pharmaceutical drugs based on their regulatory risk and abuse potential using Machine Learning techniques.  
It helps in identifying high-risk drugs by analyzing structured data and applying predictive models.

##  Project Goal
- Analyze drug-related dataset
- Perform data preprocessing and cleaning
- Build machine learning models for classification
- Predict regulatory risk category of drugs

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Flask (for deployment)

##  Features
- Data Cleaning (handling missing values)
- Feature Scaling & Outlier Treatment
- Exploratory Data Analysis (EDA)
- Model Training & Evaluation
- Drug Risk Classification Prediction
- Web Application using Flask

##  Dataset
- Pharmaceutical dataset with drug-related features
- Includes attributes related to regulatory risk and abuse potential

##  Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree

##  How to Run the Project

### 1. Clone the repository
git clone https://github.com/your-username/Drug-Risk-Classification.git


### 2. Navigate to project folder

cd Drug-Risk-Classification


### 3. Install required libraries

pip install pandas numpy scikit-learn matplotlib seaborn flask


### 4. Run the Flask App

python app.py


### 5. Open in browser

http://127.0.0.1:5000/

- The project is deployed using Flask for web-based interaction.
